# .ai/ — AI Development Pipeline

Minimal structure for AI-assisted development continuity on Incubator OS.

## Structure

```
.ai/
├── sessions/     Working memory — what happened each session
├── sprints/      Planning board — what should happen
├── skills/       Role definitions for AI agents
│   ├── angular-developer/
│   └── php-developer/
├── standards/    Shared conventions
└── README.md     This file
```

## Workflow

Every session:
1. Read current sprint
2. Read latest session
3. Understand the task
4. Work
5. Update session
6. Update sprint if needed

Every completed task updates: Code → Session → Sprint
