## Goal

Build the first business capability vertical slice: Grant Application overview. Backend exposes a single `getOverview()` endpoint instead of 6 separate CRUD calls. Frontend consumes it in one request.

- Created `api/grant-applications/queries/get-overview.php` — single endpoint returning application, workflow, bank statements, compliance, SCM verification, DD answers

- Created `services/grant-applications/GrantApplicationService.php` — business capability with `getOverview()` and `updateApplication()`
- Created `services/grant-applications/README.md` — capability documentation
- Created Angular `GrantApplicationApiService` — calls new capability endpoints
- Created `ApplicantOverview` TypeScript interface — typed contract for the read model
- Refactored `ApplicantOverviewComponent` — owns its data loading via `getOverview()`, no longer depends on `FormTemplateService` or old `GrantApplicationService`

- Refactored 8 child components (checklist, company-info, address, ownership, directors, documents, edit-modal, id-card, stage-actions) to use new `GrantApplicationApiService` for saves

- Removed `FormTemplateService` dependency from overview component
- Removed duplicate `loadWorkflow()` calls from child components

## Decisions

- **Capability endpoints** live in `api/` (new) alongside legacy `api-nodes/` (untouched)
- **Queries vs Commands** separation — `queries/` for reads, `commands/` for writes
- **Patch-based updates** — frontend sends only changed fields, backend reads current state, merges, validates, saves
- **Services receive `Node` (repository), not PDO** — endpoint handles wiring
- **Private helper methods** in services wrap `Node` calls so public methods read as business intent
- **Overview component owns its data** — shell passes only `applicantId`, overview calls `getOverview()` once
- **Child components receive pre-loaded data** via `@Input()` where possible, falling back to their own API call when needed
- **Old `GrantApplicationService`** kept for compliance and bank-statements route pages (not yet migrated)
