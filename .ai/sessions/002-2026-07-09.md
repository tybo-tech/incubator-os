## Decisions

1. **No new SQL tables** — all data stored in the generic `nodes` table with `type = 'financial_indicators'`
2. **Calculations never persisted** — `FinancialIndicatorCalculator` computes on every response
3. **March-February financial year** — annual report always returns 12 months in audit year order
4. **Duplicate month rejection** — `CreateIndicator` checks for existing company/fy/month before insert
5. **Single-use tokens** — `FinancialIndicatorLinkRepository` marks requests as completed after submission
6. **Token expiry** — links expire after 7 days, validated in `PublicSubmit`
7. **camelCase JSON** — all PHP DTOs and TypeScript interfaces use camelCase
8. **Thin controllers** — controllers only parse input and call use cases; all logic in use cases and calculator
