# Project Overview

## Incubator OS

| Layer | Technology |
|---|---|
| Frontend | Angular 19 |
| Backend | PHP 8.1 |
| Database | MySQL |
| Styling | SCSS + Tailwind CSS v4 |
| Charts | chart.js via lucide-angular |

## Architecture

- Business logic lives in backend
- Frontend is presentation only
- API base: `http://localhost:8080/`
- Data layer: NodeService (generic CRUD for JSON nodes)
